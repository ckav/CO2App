# Contributing to CO2App

## Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/CO2App.git
    ```
2.  **Open in VS Code**:
    ```bash
    code CO2App
    ```
3.  **Install Recommended Extensions**:
    *   **Live Share**: For real-time collaboration.
    *   **Prettier**: For code formatting.

## Collaboration Workflow

### Using GitHub
1.  **Pull latest changes** before starting work: `git pull`
2.  **Make changes** in your local files.
3.  **Commit and Push**:
    ```bash
    git add .
    git commit -m "Description of changes"
    git push
    ```

### Real-time Co-creation (Live Share)
1.  One person starts a **Live Share** session (click "Live Share" in status bar).
2.  Share the link with the other person.
3.  Both can edit the same file simultaneously!

### Using AI Tools
*   **Claire**: Uses GitHub Copilot in VS Code.
*   **Workmate**: Uses ChatGPT (Plus).
    *   *Tip*: Copy code from VS Code, paste into ChatGPT with your prompt, then copy the solution back to VS Code.
